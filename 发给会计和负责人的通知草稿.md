# 发给会计和负责人的通知草稿

## 邮件标题

`请协助提供研发费用明细账测试样表`

## 收件人建议

- 会计负责人
- 张英
- 需要知情的管理层

## 邮件正文

```
各位好，

我们准备把研发补贴台账升级为“自动取得账表并生成预警”的方式，目的是减少每月手工录入和人工搬运，并及时提醒杭州微新、深圳微智两边研发投入是否达到补贴要求。

第一版不会做复杂审批，也不会上传所有凭证和发票。邮件附件可以作为测试和过渡入口，但长期目标是通过会计系统接口、定时邮件或自动导出取得数据。

请会计协助提供一份真实导出的研发费用明细账测试样表，要求如下：

1. 格式：Excel 或 CSV。
2. 月份：任选最近一个已经做账完成的月份。
3. 字段尽量包含：
   - 日期
   - 法人主体
   - 会计科目
   - 摘要
   - 金额
   - 凭证号
   - 部门或项目
   - 供应商或员工
   - 备注

参考文件：

- 会计研发费用明细样表.csv
- 会计样表填写说明.md
- 测试邮件模板.md
- 邮件闭环AI原生执行原则.md

张英这边后续主要确认系统无法判断的记录，例如人员费用分摊、费用归属杭州还是深圳、费用应归到哪个补贴项目。

管理层后续只看简版 HTML 月报，不看明细表。

请先提供一份测试样表，我们用来确认字段是否能被系统识别。同时请会计或 IT 帮忙确认：当前会计系统是否支持 API、定时邮件、报表订阅或自动导出。

谢谢。
```
