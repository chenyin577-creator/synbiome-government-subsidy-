## 每次更新怎么做

最简单做法：

1. 打开 GitHub 里的这个项目。
2. 上传并覆盖这 4 个文件：
   - `index.html`
   - `app.js`
   - `styles.css`
   - `config.js`
3. 回到腾讯云 CloudBase。
4. 进入“静态网站托管”里的这个应用。
5. 点“更新服务”。
6. 等它发布完成，再打开网站检查。

## 数据怎么保存

正式腾讯云网址打开时，台账会尝试保存到腾讯云数据库。

本地电脑直接打开 `index.html` 时，只是预览，不会同步到云端，页面会显示“本机预览”。

## 日常使用

1. 项目负责人进入 **项目设置**，新增或维护项目。
2. 会计每月进入 **每月录入**，填本月新增研发投入。
3. 管理层进入 **平衡总览**，看杭州和深圳哪里还缺研发投入。
